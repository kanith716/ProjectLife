Featured case study

FaceSense

An industry game changer, FaceSense is our indigenous ML powered face recognition authentication solution. An AI application, it runs on a Convolutional Neural Network (CNN) that authenticates customers factoring in facial changes related to age. It provides real-time verification when it retrieves the KYC document on file and matches it with the live photo thereby reducing fraud.

  • Problem
    Customer verification poses many challenges when done...

    Customer verification poses many challenges when done manually at the branch for both purchase and policy payouts. Long queues, age-related facial changes and many such challenges provide room for error and fraud.
    “The identity theft and fabrication of documents fraud comprises of nearly 40% of all frauds in the BFSI sector”.

  • Solution
    Enter FaceSense, our in-house face recognition solution...

    Enter FaceSense, our in-house face recognition solution that recognizes customers at different stages of the insurance lifecycle. An ML powered face authentication solution, it runs on a convolutional Neural Network that identifies and authenticates customers through real-time verification that retrieves the KYC document on file and matches it with the live photo.

    Key Highlights:

    Developed in-house using computer vision and deep learning algorithms, FaceSense runs on a Convolutional Neural Network. Tested on 1,00,000+ test cases with a 92% accuracy and 100% precision.

    Involves image processing-based pre-processing, Video Analytics-based liveliness detection and Convolutional Neural Network (CNN) based Face Recognition.

    Available on multiple channels (Mobile, web, cloud, license, etc.) to allow verification by agents via desktop and mobile phone

    Can be scaled up to handle ~1000 concurrent requests in near real time.

    Significant error reduction and processing time and an enhanced customer experience.

  • Industry Impact
    FaceSense is a game changer as it allows for instant customer...

    FaceSense is a game changer as it allows for instant customer verification and data retrieval. It has many other uses as well: ideal for employee attendance marking, cardless ATM transactions as well as pre-populating online forms through face recognition. We are the first insurance company to verify survival status of annuitants through video. FaceSense now can also be deployed across different verticals like branch, new business login and pre-conversion verification. It can also be used to handle impersonation and fraud.
